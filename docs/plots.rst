Plots
=====

The :py:mod:`.plots` module contains plotting functions for Curveball.
Functions that plot growth data expect a :py:class:`pandas.DataFrame` generated by the :py:mod:`.ioutils` module.
Functions that plot the results of growth model fitting often expect the result of the fitting - one or more :py:class:`lmfit.model.ModelResult` objects.

Example
-------

.. plot::

   import matplotlib.pyplot as plt
   import numpy as np

   t = np.linspace(0, 10, 101)
   plt.plot(t, np.sin(t) * 0.5 + 0.5)
   plt.fill_between(t, 0.3, 0.7, alpha=0.1)
   plt.title("Synthetic growth-like curve")
   plt.xlabel("Time (hours)")
   plt.ylabel("OD")
